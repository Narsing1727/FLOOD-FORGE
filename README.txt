Flood3D is a native C++ desktop application for terrain visualization and flood simulation.

## Tech Stack
- C++ (MSVC)
- Qt 6 (Widgets)
- OpenGL
- GDAL / PROJ (OSGeo4W runtime)

## Features
- DEM-based terrain loading
- OpenGL 3D visualization
- Flood depth simulation
- Offline processing (no Python dependency)
- Portable Windows release build

## Run Instructions
1. Extract the folder
2. Double-click `run_flood3d.bat`
3. No installation required

## Status
Release build tested and working on Windows 64-bit
